# KoSpacing-tensorflow
Automatic Korean words spacing with Tensorflow

###Environments
* OS : Ubuntu 16.04/18.04 x86-64 ~
* CPU : any (quad core ~)
* GPU : GTX 1060 6GB ~
* RAM : 16GB ~
* Library : TF 1.x with CUDA 9.0~ + cuDNN 7.0~
* Python 3.x

## Pre-Requisites
* python 3.x
* java 1.7+
* tensorflow 1.x
* Mecab-ko
* gensim and KoNLPY and soynlp
* numpy
* tqdm
* h5py
